# premium-turkish-restaurant

Türkisches Restaurant-Template (Döner, Kebap, Grill) für die Website-Factory,
basierend auf dem Umay Bistro, Oftersheim. Scroll-Scrub-Video-Hero, Dish-Carousel,
Speisekarte, Reviews, Galerie und Kontaktseite.

Next.js App Router (`src/` Layout), Tailwind v4, GSAP ScrollTrigger.

Alle dynamischen Daten kommen über `src/config/locations/default.ts` aus den
factory-generierten Modulen unter `lib/` (siehe `AGENTS.md`).

## Entwicklung

```bash
npm install
npm run dev
```

Öffne http://localhost:3000.
